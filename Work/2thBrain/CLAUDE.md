# 2thBrain — Obsidian Vault

This is an Obsidian knowledge vault within a multi-vault system at `/home/leilei/Documents/Obsidian/`. It serves as the Work sub-vault under `Work/2thBrain/`.

## Vault System Architecture

| Vault | Purpose |
|-------|---------|
| **Finance** | Auto-generated market analysis reports, stock/portfolio templates |
| **News** | Daily tech news aggregation (`Daily/tech-news-YYYY-MM-DD.md`) |
| **Personal** | Self-evolution tracking, OpenClaw setup guides |
| **Main** | Primary knowledge hub |
| **Work/2thBrain** | This vault — work-related knowledge |

## Content Conventions

- **Language**: Simplified Chinese (中文) is the primary language
- **Frontmatter**: All files use YAML frontmatter with `title`, `date`, `tags` (and optionally `time`)
- **Tags**: Lowercase, hyphen-separated (e.g., `market`, `tech`, `daily`)
- **Links**: Use Obsidian `[[wiki-links]]` for cross-references; relative paths for cross-vault links
- **Callouts**: Use Obsidian callout syntax (`> [!info]`, `> [!tip]`, `> [!warning]`)
- **Templates**: Use `{{placeholder}}` Handlebars-style variables
- **Hidden comments**: Use `%%...%%` for template instructions

## OpenClaw Integration

Content across vaults is auto-generated by OpenClaw AI agents. Auto-generated files include the footer attribution `由 OpenClaw 自动生成`. Daily files follow the naming pattern `{type}-YYYY-MM-DD.md`.

## Commands

No build, lint, or test commands — this is a pure content vault.
